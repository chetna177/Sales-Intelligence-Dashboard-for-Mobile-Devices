# Sales-Intelligence-Dashboard-for-Mobile-Devices
Designed and developed a comprehensive Power BI dashboard for mobile sales analysis using Excel-based datasets. The dashboard includes:

Sales and revenue KPIs
Brand-wise and model-wise performance analysis
Region and customer insights
Time-series sales trends
Interactive filters and drill-down reports

This project demonstrates skills in data visualization, Power Query, DAX, dashboard design, and business intelligence reporting.
